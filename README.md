## 1. Main goal
The main goal of the project is to use in practice the skills related to RTOS on microcontrollers acquired during the course: "Kurs FreeRTOS na STM32 dla początkujących"

## 2. Tools
Software:
- STM32CubeIDE

Hardware:
- Nucleo STM32L476RG
- Oled display
- BMP280
- KY037
